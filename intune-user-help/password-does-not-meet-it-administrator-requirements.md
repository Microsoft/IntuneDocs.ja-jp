# <a name="device-password-requirements-for-enrolled-devices"></a>登録済みデバイスのデバイスパスワードの要件

職場または学校のリソースへのアクセスを許可する前に、より安全なパスワードを作成するように組織から要求される場合があります。 この記事では、Windows 10、iOS、macOS、および Android デバイスの一般的なパスワード要件について説明します。 これらの要件のすべてが組織によって強制されるとは限りません。  


パスワードまたはパスコードが要件を満たさなくなった場合は、ポータルサイトからメッセージを受信します。 必要な変更について説明します。 メッセージに詳細情報が提供されていない場合は、この記事を参照として使用して、現在のパスワードと比較してください。  

> [!IMPORTANT]
> 要件を満たすようにパスワードを変更したが、まだ通知を受信している場合は、デバイスを再起動します。  

詳細については、IT サポート担当者にお問い合わせください。 連絡先情報については、[ポータル サイト Web サイト](https://go.microsoft.com/fwlink/?linkid=2010980)をご確認ください。  

## <a name="windows-10-password-requirements"></a>Windows 10 のパスワード要件

| メッセージ | 修正方法 |
|-----------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| パスワードが必要です。 | パスワードを設定してください。 組織では、デバイスのロックを解除するためにパスワードを入力する必要があります。 |
| パスワードが簡単すぎます。 |  パスワードには、1234や1111のように、連続する数値や繰り返しの番号が含まれていないことを確認してください。 |
| パスワードが短すぎます。| より多くの文字を含むパスワードを更新または設定します。 組織では、パスワードが特定の長さである必要があります。 実際に何を選択するかは異なりますが、必要な最小の長さは4文字で、最大値は16です。 |
| パスワードには数字のみを含める必要があります。 | 数字のみを含むパスワードを設定します。|
| パスワードには英数字のみを使用できます。 | 数字と文字の組み合わせを含むパスワードを設定します。|
| パスワードには複雑な文字を含める必要があります。 | 数字、大文字、および `$`、`%`、`#`などの記号などの複雑な文字を追加します。 組織では、他のユーザーがパスワードを推測しづらいように、文字、数字、英数字以外の文字を組み合わせて使用する必要があります。|  
| パスワードの有効期限が切れています。 | 新しいパスワードを設定します。 組織では、一定の日数が経過した後にパスワードを変更するよう求められています。 |
| パスワードがごく最近に使用されました。 | 前に使ったことのないパスワードを選択してください。 組織では、パスワードを再利用する前に一定の時間が経過している必要があります。 |

## <a name="ios-passcode-requirements"></a>iOS のパスコード要件

| メッセージ | 修正方法 |
|-----------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| パスコードが必要です。| パスコードを設定します。 組織では、デバイスのロックを解除するためのパスコードを入力する必要があります。 |
| パスコードが単純すぎます。 |  パスコードには、1234や1111などの連続する番号または繰り返される数値が含まれていないことを確認してください。 |
| パスコードが短すぎます。 | より多くの文字でパスコードを更新または設定します。 組織では、パスコードが一定の長さであることが要求されています。 実際に何を選択するかは異なりますが、必要な最小の長さは4文字で、最大値は14です。 パスコードを変更すると、Apple から6文字以上の文字を入力するように求めるプロンプトが表示される場合があります。このメッセージは、Apple システムの推奨事項です。 組織で必要とされるのが4文字または5文字のパスコードのみである場合は、6桁のパスコードを入力する必要はありません。|  
| パスコードには数字のみを含める必要があります。 | 数字のみを含むパスコードを設定します。|
| パスコードには英数字のみを使用できます。| 数字と文字の組み合わせを含むパスコードを設定します。|
| パスコードには英数字以外の文字を含める必要があります。 | `&`、`!`、`$`、`%`、`#`などの特殊文字を追加します。 組織では、他のユーザーがパスコードを推測しづらいように、文字、数字、英数字以外の文字を組み合わせて使用する必要があります。|
| パスコードが期限切れです。 | 新しいパスワードを設定します。 組織では、一定の日数が経過した後にパスワードを変更するよう求められています。 |
| パスコードが最近使用されました。| 前に使ったことのないパスコードを選択してください。 組織では、パスコードを再利用する前に一定の時間が経過している必要があります。 |
|タッチ ID または顔 ID 認証が必要です。 | タッチ ID または顔 ID を設定します。 パスワードまたはクレジットカード情報にオートコンプリートを使用する前に、これらの方法のいずれかを使用して認証を行う必要があります。 |

## <a name="android-password-requirements"></a>Android パスワードの要件
| メッセージ | 修正方法 |
|-----------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| パスワードが必要です。 | パスワードまたは PIN を設定します。 組織では、デバイスのロックを解除するためにパスワードを入力する必要があります。 |
| パスワードが簡単すぎます。 |  パスワードまたは PIN に、1234や1111などの連続する番号または連続する数値が含まれていないことを確認します。 |
| パスワードが短すぎます。 | より多くの文字を含むパスワードを更新または設定します。 組織では、パスワードが特定の長さである必要があります。|
| パスワードに数字を含める必要があります。 | 数字を含むパスワードまたは PIN を設定します。|
| パスワードに文字を含める必要があります。 | アルファベットの文字を含むパスワードを設定します。|
| パスワードには英数字を含める必要があります。 | 数字と文字の組み合わせを含むパスワードを設定します。|
| パスワードには、英数字と記号を含める必要があります。 | 文字、数字、および特殊文字 (`&`、`!`、`$`、`%`、`#`など) が混在するパスワードを設定します。 |
| パスワードは生体認証テクノロジを使用する必要があります。| 指紋や顔認識などの生体認証を使用するようにデバイスを設定します。
| パスワードの有効期限が切れています。 | 新しいパスワードを設定します。 組織では、一定の日数が経過した後にパスワードを変更するよう求められています。 |
| パスワードがごく最近に使用されました。 | 前に使ったことのないパスワードを選択してください。 組織では、パスワードを再利用する前に一定の時間が経過している必要があります。 |
